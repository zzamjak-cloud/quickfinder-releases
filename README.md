# QuickFinder Releases

QuickFinder 데스크톱 앱의 릴리스 배포 저장소입니다.

- 설치 파일과 자동 업데이트 피드(`latest.json`)가 [Releases](../../releases)에 게시됩니다.
- 소스 코드는 별도 비공개 저장소에서 관리됩니다.
- 구매: Gumroad (링크 추후 추가)

## 포함 릴리스

- `v*` — 앱 릴리스 (macOS `.dmg`/`.app.tar.gz`, Windows NSIS `.exe` + 서명 파일, `latest.json`)
- `portable-tools-v1` — 폰트 병합 기능용 Python+fontTools 포터블 패키지 (PSF/MIT 라이선스)
